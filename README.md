# Financial Portfolio Analyzer

A Python-based quantitative finance project that analyzes stock portfolios using real-world market data and advanced risk metrics.

## Features

- Historical stock analysis
- Daily and cumulative returns
- Annual volatility
- Sharpe Ratio
- Correlation matrix visualization
- Monte Carlo simulation
- Value at Risk (VaR)
- CAPM expected returns
- Portfolio optimization
- Efficient Frontier analysis
- Excel and PDF report generation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- yfinance
- OpenPyXL
- ReportLab
